# ☕ Arcadia Speciality Coffee House

> A premium, responsive, and modern website for **Arcadia Speciality Coffee House** located in Katpadi, Vellore (near VIT). Designed with glassmorphism aesthetics, smooth animations, and a delightful user experience.

Live Website: **[https://jayadhanush871.github.io/arabic-coffee-website/](https://jayadhanush871.github.io/arabic-coffee-website/)**

---

## ✨ Features

- **Premium Glassmorphism UI**: Beautiful, modern design with subtle blur effects, gradients, and custom typography using the *Inter* and *Playfair Display* fonts.
- **Fully Responsive**: Optimized for all devices—desktops, tablets, and smartphones—with an intuitive mobile drawer menu.
- **Interactive Menu**: Tabbed menu system showing prices and categories ranging from Hot & Artisan Coffee to Pastas, Burgers, and Desserts.
- **Google Maps Integration**: Direct maps embed and a button to get turn-by-turn directions to the coffee house.
- **Interactive Feedback Form**: Built-in review and inquiry form with a dynamic star-rating system.
- **Testimonials Section**: Highlights reviews from Google Local Guides and guests.

---

## 🛠️ Built With

- **HTML5**: Semantic tags for clean structure and better SEO.
- **Vanilla CSS3**: Tailored styles, grid & flexbox layouts, custom keyframe animations, and glassmorphic panels.
- **JavaScript (ES6)**: Vanilla JS for mobile nav toggling, interactive menu category switching, form handling, and star-rating behavior.
- **Font Awesome**: Premium icons for a polished aesthetic.

## 🚀 How to Run Locally

Since this is a static website, you can run it directly without installing complex dependencies:

1. **Clone the repository:**
   ```bash
   git clone https://github.com/jayadhanush871/arabic-coffee-website.git
   cd arabic-coffee-website
   ```

2. **Open `index.html`** in any web browser, or start a local server:
   - **With Python:**
     ```bash
     python -m http.server 8000
     ```
     Then open `http://localhost:8000` in your browser.
     
   - **With VS Code Live Server Extension:**
     Right-click `index.html` and select **"Open with Live Server"**.

---

## 🌐 Deploying to GitHub Pages

The website is configured to run out-of-the-box on GitHub Pages. To activate it on your repository:

1. Go to your repository settings on GitHub.
2. Select **Pages** from the left sidebar.
3. Under **Build and deployment**, set the source branch to **`main`** and the folder to **`/ (root)`**.
4. Click **Save** and wait about a minute for the build to finish!
